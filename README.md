# Font-Thai-for-Retroarch
stock+thai system font retroarch

โหลดไฟล์ฟอนต์ mplus-1p-regular.ttf

ขั้นตอนการติดตั้ง
นำไฟล์ฟอนต์เข้าเครื่อง: คัดลอกไฟล์ฟอนต์ไปวางไว้ในโฟลเดอร์ assets/pkg/fonts หรือโฟลเดอร์ system ภายในไดเรกทอรีของ RetroArch
*ถ้าไม่คัดลอกไปวางที่system สามาถตั้งโฟลเดอร์ปลายทางไปยังโฟลเดอร์อื่นที่เราเก็บไฟล์ได้

ตั้งค่าในตัวโปรแกรม:

ไปที่ Settings (การตั้งค่า)

เลือก User Interface (ส่วนติดต่อผู้ใช้)

เลือก Appearance (รูปลักษณ์)

เปลี่ยนฟอนต์เมนู:

หาหัวข้อ Menu Font (ฟอนต์เมนู)

เลือกไปที่ไฟล์ฟอนต์ mplus-1p-regular.ttf ที่คุณนำไปวางไว้

เปลี่ยนภาษาของระบบ:

กลับไปที่ Settings > User (ผู้ใช้) > Language (ภาษา)

เลือกเป็น Thai (ภาษาไทย)

ข้อควรระวัง
Menu Driver: หากใช้ไดรเวอร์เมนูแบบ Ozone (หน้าตาคล้าย Switch) หรือ XMB (หน้าตาคล้าย PS3) จะรองรับการเปลี่ยนฟอนต์ได้ดีที่สุด
หากใช้ไดร์เวอร์เมนูอื่น ต้องเอาไฟล์ mplus-1p-regular.ttf ไปทับ โดยเปลี่ยนชื่อตามไฟล์font ไดร์เวอร์เมนูนั้นๆ ซึ่งอาจจะมี1-3ไฟล์ต้องทำให้ครับ
